# InviteWedding

This is a monorepo for managing apps related to wedding invitations.

## Structure
- `apps/`
  - `frontend/`: Contains the frontend application
  - `backend/`: Contains the backend application
- `packages/`: Contains shared utilities and components

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ``` 

2. Run the applications:
   - Frontend: `npm start frontend`
   - Backend: `npm start backend`
